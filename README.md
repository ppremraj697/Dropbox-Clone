This is a Dropbox Clone project in [Next.js](https://nextjs.org/) bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## Key Skills
React/NextJS(Version 14), Firebase and Cloud Firestore(BaaS), Tailwind CSS, shadcn/ui, Zustand, Clerk, Vercel.
